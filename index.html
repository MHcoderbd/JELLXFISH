Wir implementieren vernetzte Embedded Software.

Unser service

Wir sind ein Startup an der TU München und entwickeln Software für Embedded Systems. Insbesondere fokussieren wir uns auf FPGA-SoCs, Linux, RISC-V, Amazon IoT Cloud und Embedded Security. Unser Dienstleistungsportfolio ist unten aufgeführt. Weitere Informationen werden angezeigt, indem Sie den Mauszeiger über das entsprechende Symbol bewegen oder es anklicken.

FPGA-SoC

FPGA-SoC

Sie möchten einen autonomen Roboter bauen oder Machine Learning an der Edge betreiben? Wir entwickeln ein speziell auf Ihre Bedürfnisse zugeschnittenes Linux- und FPGA-SoC-System.

VHDL/Verilog

VHDL/Verilog

Wir entwickeln Hardwarebeschleuniger für Ihre Anforderungen in VHDL oder Verilog. Zum Beispiel: Neuronale Netze, kryptografische Beschleuniger oder benutzerdefinierte AXI-Module.

Pentesting

Pentesting

Wir testen Ihre eingebetteten Systeme auf Schwachstellen und identifizieren Härtungsmaßnahmen. Wir führen Angriffe über Hardware (z. B. JTAG) oder Software durch.

RISC-V

RISC-V

Sie möchten Ihren eigenen RISC-V Prozessor als FPGA-Soft-CPU? Wir bauen Ihren eigenen RISC-V Prozessorsystem in VHDL/ Verilog und erstellen benutzerdefinierte Software.

AWS Cloud

AWS Cloud

Wir integrieren Ihre eingebetteten Systeme/ IoT-Geräte in die AWS Cloud. Darüber hinaus entwerfen wir Cloud-Appliances nach Ihren Anforderungen.

Cybersecurity

Cybersecurity

Wir sehen Cybersicherheit als integralen Bestandteil unserer Softwareentwicklungsmethoden. Wir verwenden sichere Codierungstechniken, um die Oberfläche von Cyberangriffen zu reduzieren.

RUST

RUST

Sie möchten verstehen, welche Vorteile RUST bietet? Wir portieren Ihre vorhandene Software auf RUST und führen Benchmarks für Sie durch.

C/Python

C/Python

Wir entwickeln Software in C und Python für Mikrocontroller, Embedded Linux und FPGA-SoCs.

Cloud-OS / loT

Cloud-OS / loT

Wir helfen beim Aufbau eines sicheren Cloud-Betriebssystems für Ihre eingebetteten Systeme / IoT-Geräte. Verwalten und orchestrieren Sie Ihre Geräte über ein Cloud-Dashboard.

Andriod Apps

Andriod Apps

Bringen Sie Ihre industriellen Steuergeräte auf die nächste Stufe. Wir portieren Ihre vorhandene Industriesoftware auf ein Android App/ Tablet-System und bieten Ihnen die Möglichkeit, Ihre Appliances remote zu verwalten.

Security Lifecycle

Security Lifecycle

Wir führen Sicherheitsanalysen durch, übernehmen die Projektleitung für Sie, programmieren nach Secure-Coding Prinzipien und entwickeln Sicherheitsarchitekturen für Embedded Systems.

Embedded Linux

Embedded Linux

Unsere Stärke liegt in der Programmierung Embedded Linux-Systeme für den industriellen Einsatz. Wir entwickeln Kernelmodule, User-Space-Anwendungen und Netzwerk-Appliances in C und Python.

Unsere Beispielprojekte

RISC-V Edge Computing.

Wer das iPhone der Autoindustrie bauen will, sollte von Apple und Tesla lernen und eigene Prozessoren bauen.
Wir entwickeln für Sie ein RISC-V basierte FPGA Soft-CPU. In eigenen Hardwarebeschleunigern können Sie Machine Learning oder Krypto-Algorithmen auslagern und Kunden ein einzigartiges Hardware-Software Ökosystem bieten.

Vom Sensor in die Cloud.

Was wäre, wenn Sie den Status jedes IoT-Geräts, jede einzelne Sekunde wissen würden? Sie könnten diese Daten verwenden, um Ihr Produkt noch weiter den individuellen Bedürfnissen Ihres Kunden anzupassen.
Wir integrieren Ihre IoT-Geräte in die AWS Cloud und erstellen Kontrolldienste wie ein automatisiertes Device und Security Management.

Embedded Security.

Besondere Operationen wie kryptographische Schlüsselverwaltung, Authentifizierung und Ausführung von kritischen Anwendungen sollten in einer besonders geschützten Umgebung betrieben werden.
Wir helfen gerne in der Sicherheitsanalyse Ihrer Embedded Systems. Wir implementieren Trusted Execution Environments, Device Authentication Lösungen, Secure Boot und Secure Update Lösungen.

Security Management.

Klassische Sicherheitsansätze, die nur auf dem Perimeter-Schutz basieren und vor allem auf Prävention setzen, haben sich weitgehend als unzureichend erwiesen. Gefragt sind neue ganzheitliche Lösungskonzepte, die den gesamten End-to-End-Sicherheitsservice abdecken sowie ein umfassendes und durchgängiges Risikomanagement sicherstellen.
Wir übernehmen die Projektleitung und setzten das IoT-Security-Lifecycle-Modell bei Ihrem Projekt um.
